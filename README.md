# 🛒 Ecommerce Store

## 📌 Project Description
A full-stack Ecommerce Store built using HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, and Mongoose.

## 🚀 Features
- User Registration & Login
- Product Listing
- Shopping Cart
- MongoDB Database Integration
- Responsive UI

## 🛠️ Technologies Used
- HTML5, CSS3, JavaScript
- Node.js, Express.js
- MongoDB, Mongoose
